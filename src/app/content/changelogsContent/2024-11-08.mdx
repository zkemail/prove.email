## ZK Email Updates
New Registry Development https://github.com/zkemail/registry
We're preparing to announce our new SDK Registry platform, designed to improve the developer experience. The Registry will feature GitHub authentication, blueprint management, and an intuitive UI. We've also integrated Google login for email fetching and implemented a Claude-powered API endpoint that assists in blueprint field generation

Clave Account Recovery https://github.com/zkemail/clave-email-recovery
We have added new security features to the Clave Account Recovery system:
- Kill switch functionality for emergency scenarios
- New UserOverridableDKIMRegistry management scripts
- Updated dependencies and streamlined package versions
- Comprehensive audit fixes implementation

JWT TX Auth Updates https://github.com/zkemail/jwt-tx-auth
The JWT TX Auth system has evolved to support more flexible authentication scenarios. We've introduced anonymous domain support backed by Merkle tree verification, while optimizing the domain extraction process.

Noir Updates https://github.com/zkemail/zkemail.nr
Significant progress has been made in our Noir implementation, particularly in the quoted-printable decode functionality. Also the code audit has started. The team has:
- Implemented JavaScript circuit simulation
- Improved test vector handling
